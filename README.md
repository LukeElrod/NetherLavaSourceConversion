# NetherLavaSourceConversion

NetherLavaSourceConversion is a simple mod that allows lava to act like water only in the nether. This means that you can make infinite source pools of lava in the nether but not in the overworld.

The vanilla mojang `lavaSourceConversion` will make lava act like lava EVERYWHERE. This mod makes the source conversion only work in the nether.

## Requirements
- neoforge 1.21.1
